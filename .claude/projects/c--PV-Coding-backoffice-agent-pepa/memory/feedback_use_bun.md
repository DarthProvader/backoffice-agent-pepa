---
name: use-bun-not-npm
description: User prefers bun over npm for package management and running scripts
type: feedback
---

Use `bun` instead of `npm` for all package management and script running.
- `bun install` instead of `npm install`
- `bun run dev` instead of `npm run dev`
- `bunx` instead of `npx`

**Why:** User preference, bun is faster.
**How to apply:** Always use bun commands throughout the project.
